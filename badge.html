Background colors

Use Bootstrap’s background utility classes to quickly change the appearance of a badge. Please note that when using Bootstrap’s default CSS classes using the DITA @outputclass attribute with the value bg-light, you’ll likely need a text color utility attribute on @outputclass like text-dark as well for proper styling. This is because background utilities do not set anything but background-color.
