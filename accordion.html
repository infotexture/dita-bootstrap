Accordion

Build vertically collapsing accordions in combination with Bootstrap’s Collapse JavaScript plugin.

How it works

The accordion uses collapse internally to make it collapsible. To render an accordion that’s expanded, add the class open in addition to accordion to the @outputclass.

Flush

Add the class accordion-flush to @outputclass to remove the default background-color, some borders, and some rounded corners to render accordions edge-to-edge with their parent container.
